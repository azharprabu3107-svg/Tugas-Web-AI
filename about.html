Tujuan Pembuatan Website

Website ini dibuat sebagai tugas akhir mata kuliah Pemrograman Berbasis Web dengan tujuan:

  • Menerapkan konsep HTML5 semantik dalam struktur website
  • Mengimplementasikan CSS3 modern untuk desain responsif
  • Mengintegrasikan multimedia (video, audio, gambar) dalam website
  • Membuat demo interaktif sederhana menggunakan JavaScript
  • Menyajikan konten edukatif tentang AI dan chatbot
