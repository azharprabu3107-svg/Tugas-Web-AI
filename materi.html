Elemen Semantik HTML

Website "EduBot AI" menggunakan elemen semantik HTML5 untuk struktur yang lebih bermakna dan aksesibel.

Dengan menggunakan elemen Semantik dapat membantu mesin pencari dan screen reader untuk memahami struktur dan konten dari halaman web dengan maksimal. Berikut elemen-elemen yang digunakan:

Elemen HTML Fungsi Contoh Penggunaan
<header> Bagian atas website atau kepala kop dari web (logo, navigasi) Header dengan logo dan menu navigasi
<nav> Navigasi utama website Menu untuk berpindah antar bagian
<main> Konten utama website Seluruh konten edukasi tentang AI
<section> Pengelompokan konten atau membuat bagian artikel Bagian Hero, Dasar AI, Video AI
<article> Konten independen yang utuh Penjelasan tentang dasar-dasar AI
<aside> Konten sampingan/tambahan pada bagian samping Sidebar dengan fakta menarik AI
<footer> Bagian bawah website Informasi hak cipta dan link
<figure> & <figcaption> Gambar dengan keterangan Ilustrasi AI dengan penjelasan

Contoh Kode Struktur Semantik:

<!-- Struktur Dasar Website -->
<header>
  <nav>...</nav>
</header>

<main>
  <article id="basics">
    <h2>Dasar-dasar AI</h2>
    <p>Penjelasan tentang AI...</p>
  </article>
</main>

<footer>
  <p>Hak Cipta Â© 2025</p>
</footer>

Elemen Multimedia HTML

Selain menampilkan gambar statis atau diam, HTML juga dapat menampilkan gambar bergerak dan juga bersuara alias video. Website ini menggunakan berbagai elemen multimedia untuk menyampaikan konten edukasi mengenai Chatbot AI:

Elemen <video>

Menampilkan video penjelasan tentang AI dengan kontrol play, pause, dan volume. Untuk menampilkan video pada HTML dapat menggunakan tag <video>, tag ini berisi sumber video yang ingin di tampilkan.

Elemen <audio>

Menampilkan konten audio seperti podcast atau penjelasan tentang AI. Bukan hanya video saja, HTML juga dapat memutar suara atau audio menggunakan tag <audio>

Elemen <img>

Menampilkan gambar ilustrasi AI, diagram neural network, dan foto profil agar website yang dibuat tidak terkesan membosankan

Elemen <track>

Menambahkan subtitle pada video untuk aksesibilitas

Contoh Kode Multimedia:

<!-- Video dengan kontrol dan subtitle -->
<video controls poster="thumbnail.jpg">
  <source src="videochatbot.mp4" type="video/mp4">
  <track src="subtitles-id.vtt" kind="subtitles" srclang="id" label="Indonesia">
  Browser Anda tidak mendukung elemen video.
</video>

<!-- Audio player -->
<audio controls>
  <source src="podcastchatbot.mp3" type="audio/mpeg">
</audio>

CSS: Mendesain Website EduBot AI

CSS (Cascading Style Sheets) adalah bahasa pemrograman berbasis aturan yang dikembangkan oleh World Wide Web Consortium (W3C) pada tahun 1996. Secara teknik CSS bukanlah elemen wajib tetapi halaman web akan kosong tanpa gaya yang modern sehingga CSS berfungsi untuk mendesain tampilan website.

Cara kerja CSS menggunakan bahasa Inggris berbasis syntax yang dilengkap dengan rule. Kemudian bagaimana cara untuk men-style paragraf? CSS memiliki struktur syntax yang sederhana. Struktur tersebut memiliki selector dan declaration block. Sehingga pilih elemen yang ingin di style, kemudian deklarasikan elemen tersebut. Berikut dasar-dasar CSS yang digunakan dalam website ini.

Struktur Dasar CSS:

css-demo {
  color: black;
  font-weight: bold;
  font-size: 15px;
  text-align: center;
}

Contoh diatas menyatakan, semua elemen css-demo akan ditandai dengan warna hitam, huruf tebal / bold, ukuran font 15px dan tata letak atau posisinya ditengah (align-center).

CSS dan Styling Website

CSS (Cascading Style Sheets) adalah teknologi yang digunakan untuk menghias atau memperindah halaman website yang dibuat dengan HTML. Dengan CSS website yang dibuat tidak akan membosankan. Website menggunakan CSS modern untuk desain yang responsif dan menarik:

5 Macam Selektor CSS yang Digunakan pada Website "EduBot AI":

1. Selektor Universal

* { margin: 0; padding: 0; }

Reset semua margin dan padding

2. Selektor Elemen

body { font-family: 'Segoe UI'; }

Menerapkan style ke elemen body

3. Selektor Class

.content-card { background: white; }

Menerapkan style ke elemen dengan class tertentu

4. Selektor ID

#basics { animation: fadeIn; }

Menerapkan style ke elemen dengan ID tertentu

5. Selektor Atribut

[data-type] { border: 1px solid; }

Menerapkan style berdasarkan atribut

Fitur CSS Modern yang Digunakan:

  • CSS Grid untuk tata letak dua dimensi melalui kolom dan baris pada layout utama (main dan sidebar)
  • Flexbox untuk mengatur layout elemen secara fleksibel dalam container pada navigasi dan komponen kecil
  • CSS Variables menyimpan dan menggunakan kembali nilai properti CSS untuk konsistensi warna dan ukuran
  • Media Queries untuk membuat desain web responsif
  • Transisi dan Animasi untuk menciptakan interaktivitas perubahan visual yang halus dan terkontrol
  • Box-shadow untuk efek kedalaman atau bayangan pada elemen HTML
  • Border-radius untuk sudut melengkung
  • Gradients menciptakan transisi warna yang mulus

Contoh Kode CSS Grid:

/* Layout utama dengan Grid */
main {
  display: grid;
  grid-template-columns: 1fr 300px;
  gap: 3rem;
  margin-bottom: 4rem;
}

/* Responsive design untuk mobile */
@media (max-width: 992px) {
  main {
    grid-template-columns: 1fr;
  }
}

Teknik Responsive Design

Website dirancang untuk tampil optimal di berbagai perangkat:

Breakpoint yang Digunakan:

Desktop

> 992px

Layout 2 kolom (main + sidebar)

Tablet

768px - 992px

Layout 1 kolom, navigasi horizontal

Mobile

< 768px

Navigasi vertikal, font size lebih kecil

Teknik Responsive yang Diterapkan:

/* Mobile First Approach */
.container {
  width: 90%;
  max-width: 1200px;
  margin: 0 auto;
}

/* Media Query untuk Tablet */
@media (max-width: 992px) {
  main {
    grid-template-columns: 1fr;
  }
}

/* Media Query untuk Mobile */
@media (max-width: 768px) {
  .header-content {
    flex-direction: column;
  }
  nav ul {
    flex-wrap: wrap;
    justify-content: center;
  }
  .hero h1 {
    font-size: 2.2rem;
  }
}

Tips Responsive Design:

  • Gunakan unit relatif (%, rem, em) daripada px
  • Gunakan max-width untuk gambar dan kontainer
  • Uji website di berbagai ukuran layar
  • Pertimbangkan orientasi portrait dan landscape
