Pertanyaan Umum tentang Proyek

Apakah website ini menggunakan backend/server?

Tidak, website ini adalah static website yang hanya menggunakan HTML, CSS, dan JavaScript di sisi klien. Semua interaksi dilakukan di browser tanpa perlu server backend.

Bagaimana cara kerja form kontak jika tidak ada server?

Form kontak menggunakan JavaScript untuk memvalidasi dan menampilkan data yang diinput. Data ditampilkan di console browser dan alert box sebagai simulasi. Tidak ada data yang benar-benar dikirim ke server.

Teknologi apa saja yang digunakan dalam website ini?

Website ini dibangun menggunakan:

  • HTML5 untuk struktur semantik
  • CSS3 untuk styling dan responsive design
  • JavaScript untuk interaktivitas
  • Font Awesome untuk ikon
  • Google Fonts untuk tipografi
